# sandbox
This repo has been created to checkin random projects to share and collaborate with other team members.

The idea will be to create branches out of this repo, and put your code there. We will not allow anyone to make changes to main.

This will be the preferred model.
So for example is user rohit has to share some code about an experimental project with others, he should create 
* a branch on which this project will reside
* a top level folder rohit, and inside that the project directory.
* now checkin the code and share the code with others.

Once some of the projects become bigger/approved, we can allow them to be moved to other repos or a new repo of itself.
